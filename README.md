# 안심식당
<br>
기간 : 2020.09.14 ~ 2020.12.05
<br><br>
코로나19로 음식점을 이용하기 꺼려하는 소비자들을 위해 농림축산식품부에서 제공하는 공공데이터(OpenAPI)를 활용하여 만든 웹 사이트 입니다.
<br><br>

* 적용기술 
  * Back-End  
    * Spring Framework 
    * JAVA 
  * DevOps 
    * ORACLE 
  * Front-End 
    * HTML 
    * CSS 
    * Javascript 
    * BootStrap 
  * 기타 
    * 경기도 공공데이터 API 
    * Kakao MAp API
